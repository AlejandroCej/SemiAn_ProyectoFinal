# SemiAn_ProyectoFinal

Programa Parkinsee para el análisis y diagnóstico de parkinson en pacientes por medio de videos pregrabados de su marcha de forma lateral.
El código genera un esqueleto de la persona marcando de forma automatica puntos clave como articulaciones y cabeza, permitiendo cuantificar puntos visuales clave relacionados a la enfermedad, como serían inclinación de cadera y poco balanceo de brazos en la caminata.
Para usarse en google colab basta con subir un archivo con el nombre "MarchaParkinson.mp4" y correr cada celda de código.
En caso de usarse en entorno local se necesitará de cambiar la ruta de archivo de video a donde se enucuentra este en su entorno.
